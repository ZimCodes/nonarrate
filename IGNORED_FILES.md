# Ignore Files

Placed here are a list of files and folders _nonarrate_ will ignore by default.

## Invalid Folders

|           |          |                 |          |
|:---------:|:--------:|:---------------:|:--------:|
|    tl     |   menu   |       gui       |  saves   |
|  images   |  cache   |      fonts      | voice(s) |
| functions |  music   |      audio      | gallery  |
| sound(s)  | video(s) |     shaders     |   font   |
| movie(s)  | screens  | python-packages | Gallery  |
|    sfx    |   libs   |       ui        |    UI    |
| replay(s) |    bg    |       bgm       |   anim   |
|    vi     |    vo    |       img       |  camera  |

## Invalid Files
